Установка nodejs:
1. Заходим на сайт https://nodejs.org/en/
2. Скачиваем LTS версию
3. Устанавливаем

Установка зависимостей:
npm install

Запуск:
npm start

Примеры:
- http://localhost:8079
- http://localhost:8079/fines
